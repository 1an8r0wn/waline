# 介绍
![](https://img.shields.io/npm/v/@waline/client?color=blue&logo=npm&style=flat-square)   

Waline - 一款从 [Valine](https://valine.js.org) 衍生的带后端评论系统。可以将 Waline 等价成 With backend Valine.

## 特性

- 快速
- 真·安全
- Markdown 语法支持
- 轻量易用
- 免费部署
